# csesa-webd-ps
This is a static HTML + CSS clone of the CSES (Code Submission Evaluation System) homepage layout.
It visually mimics the original CSES design, with the same header, color blocks, and structure.

features of this website:


1. external login page
2. use of 2*2 grid
3. Responsive Design:
Uses a CSS grid layout that automatically stacks panels vertically on smaller screens (phones/tablets).
